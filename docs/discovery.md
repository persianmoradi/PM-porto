# Phase 2 — Discovery (working doc)

_Last updated: 2026-06-25. Content section pending Persi's input._

## Strategy (locked)

- **Primary positioning:** Frontend / design engineer. The site itself is the headline
  proof of skill. Not hedging across multiple identities.
- **AI angle:** Woven into the work, not the headline. Show AI inside projects.
- **Primary goal / CTA:** "Hire me" — full-time or part-time alongside studies, at
  startups. Freelance/consulting is a soft secondary path. One clear contact conversion.
- **Aesthetic:** Refined-minimal **canvas** with a few **bold, signature moments**
  (hero, key transitions). References: studio/agency + Awwwards SOTD energy, expressed
  with restraint. Tone: confident & polished, employer-safe.
- **Motion:** Signature, targeted interactions executed flawlessly; calm everywhere else.
  Must respect `prefers-reduced-motion`.
- **Timeline:** ASAP (weeks). Ship a tight, excellent v1 + a content roadmap for pieces
  still being built.

## Design implications

- Minimalism means substance can't hide — every section must earn its place. Few pages,
  high polish.
- "Bold moments" budget: pick ~2-3 (likely hero + one project reveal + page transitions).
  Everything else stays quiet so those land.
- Because some work is "potential," architecture must make adding case studies trivial
  (content-driven project entries), and v1 should look complete with the pieces we have.
